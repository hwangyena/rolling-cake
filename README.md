## 사용 기술

**Client**

- Next.js 13v
- Jotai, SWR
- postcss, tailwindcss
- react-three-fiber, react-three-drei

<br/>

**Server**

- [Nextjs Route Handlers](https://nextjs.org/docs/app/building-your-application/routing/route-handlers)
- Mongodb
- prisma

<br/>

**Deploy**

- AWS EC2 + Route53 + Docker
